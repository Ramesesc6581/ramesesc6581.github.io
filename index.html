⬇️ 下载最新固件 (访问发布页)

欢迎使用 FlipperZero-PhantomCN-Firmware。本固件专为中国 Flipper Zero 用户打造。它基于 Momentum 固件 (mntm-012)，并做了全面汉化。

您不再需要面对英文菜单。开机即用全中文界面。

本固件内置多项实用工具与信号包。包括 Bad Apple 动画、BLE 攻击、Sub-GHz 暴力破解与常用信号库。

📋 快速开始

您只需要花费几分钟即可完成安装。您的 Flipper Zero 会变成一个功能更强大的中文设备。

🔧 前提条件

  • 一台运行 Windows 7 或更高版本的电脑
  • 一个 Flipper Zero 设备 (任何版本均可)
  • 一根 USB 数据线 (用于连接电脑与 Flipper Zero)
  • 您的 Flipper Zero 电池电量充足 (建议 50% 以上)

⚙️ 安装步骤

  1. 访问下载页面
    点击顶部蓝色按钮，访问发布页面。
    您将被带到 GitHub 的 Releases (发布) 页面。

  2. 选择最新版本
    在发布页面中，找到最新版本的固件文件。文件名通常类似 phantom-cn-firmware-x.x.x.zip。
    点击该文件即可开始下载。

  3. 解压文件
    下载完成后，找到 .zip 文件。
    右键点击它，选择“解压缩到当前文件夹”或使用您常用的解压软件。

  4. 进入 DFU 模式 (更新模式)
    在您的 Flipper Zero 上，同时按住 左键 和 返回键。
    保持按住，然后插入 USB 数据线连接电脑。等待屏幕变为蓝色，并显示 DFU 模式图标。

  5. 运行更新脚本
    打开解压后的文件夹。找到 flash.bat 或 update.bat 文件。
    双击运行它。脚本会自动识别您的设备并刷入固件。
    注意：请勿在刷写过程中断开连接或关闭电脑。

  6. 等待完成
    脚本会在命令行窗口中显示进度。当提示“完成”或“Success”时，您可以拔掉数据线。
    您的 Flipper Zero 将自动重启，并显示全新的中文界面。

💡 主要功能

这个固件不只是换个语言。它还加入了很多新能力。

🌐 全中文界面与字体

所有菜单、设置、提示均为简体中文。固件内置宋体字库，中文字体显示清晰美观。不再有英文乱码或拼音菜单。

🎶 Bad Apple 桌面动画

固件内置经典的 Bad Apple 动画。您可以在桌面或屏幕保护模式中播放它。为您的设备增添一份趣味。

⚡ BLE Killer 工具

内置蓝牙攻击模块。可以扫描并干扰附近的蓝牙设备。专为安全研究设计。

📡 Sub-GHz 暴力破解 (PT2262)

针对常见的 PT2262 编码协议，提供自动暴力破解功能。可用于测试车库门、遥控器等的安全性。

📶 Sub-GHz 信号包

固件预装了多种实用的 Sub-GHz 信号库，包括射频干扰信号、特斯拉线圈信号、常见传感器信号等。开箱即用，无需自行捕获。

🛡️ 射频干扰与特斯拉信号

包含专门的 Sub-GHz 信号包，用于测试射频环境稳定性。支持特斯拉线圈模拟信号。

❓ 常见问题

问：刷机会损坏我的设备吗？

刷机过程非常成熟。只要您按照步骤操作，在更新过程中不拔线，一般不会出问题。如果遇到意外，您可以重新进入 DFU 模式重新刷写。

问：我可以在 macOS 或 Linux 上刷机吗？

本指南基于 Windows 编写。如果您使用其他系统，需要自行使用命令行工具 (如 dfu-util) 进行刷写。我们强烈建议您使用 Windows 以获得最佳体验。

问：安装后如何恢复原厂固件？

您可以从 Flipper Zero 官方网站下载官方固件，然后使用同样的 DFU 模式刷回。过程完全相同。

问：有些功能我不会用怎么办？

您可以查看 Issues 页面或参与社区讨论。提出具体问题，其他用户或开发者会为您解答。
